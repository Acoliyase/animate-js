Animate-JS — Static Demo (GitHub Pages)

This demo uses only the Web Animations API — no private packages required.

Example usage (vanilla)

// Uses Web Animations API
document.getElementById('box').animate([
  { transform: 'translateY(0) scale(1)', opacity: 1 },
  { transform: 'translateY(-40px) scale(1.05)', opacity: 0.9 },
  { transform: 'translateY(0) scale(1)', opacity: 1 }
], { duration: 600, easing: 'cubic-bezier(.22,.9,.34,1)' });
